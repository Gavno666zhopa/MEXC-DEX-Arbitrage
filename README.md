# MEXC-DEX-Arbitrage
# 🚀 MEXC & DEX Arbitrage Alpha-Bot 

![Status](https://img.shields.io/badge/Status-Stable-green) ![Version](https://img.shields.io/badge/Version-2.4.0-blue) ![MEXC](https://img.shields.io/badge/Exchange-MEXC-orange)

Профессиональное решение для автоматизации межбиржевого арбитража между DEX (Uniswap, PancakeSwap, Raydium) и биржей MEXC.

---

## ⚡️ Ключевые возможности

*   **Сканер 24/7:** Автоматический поиск спредов в сетях Ethereum (ERC-20), BSC (BEP-20) и Solana.
*   **Исполнение через API:** Бот моментально выставляет ордера на MEXC, когда находит профитную связку.
*   **Защита от MEV:** Использование приватных узлов (RPC) для скрытия транзакций в мемпуле.
*   **Smart Calc:** Автоматический учет Gas Fee, торговых комиссий и проскальзывания (slippage) перед входом.

## 🛠 Как начать?

1.  **Зайдите на сайт:** Перейдите по ссылке [твойсайт.com](https://твойсайт.com).
2.  **Выберите версию:** Скачайте дистрибутив под вашу ОС (Windows/Linux).
3.  **Конфигурация:** Вставьте свои API-ключи от MEXC в файл `config.yaml` (ключи хранятся локально на вашем ПК).
4.  **Запуск:** Запустите `start.exe` и бот начнет мониторинг сетей.
